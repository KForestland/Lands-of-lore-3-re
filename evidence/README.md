# Evidence

Evidence indexes and witness notes for the public LoL3 checkpoint.

Current public evidence entry point:

- [`lol3-witness-map.md`](lol3-witness-map.md)
  - short map of the main witnesses and what each one is good for

This folder should contain:

- witness maps
- short evidence indexes
- trace-reference summaries

Not included:

- raw large trace dumps
- private scratch ledgers
- large unpublished analysis ledgers

Working rule:

- keep evidence notes short, scoped, and tied back to promoted docs
- this folder should help readers navigate the checkpoint, not recreate the full private lab book
