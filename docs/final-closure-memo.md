# Final Closure Memo

This repo captures the current public LoL3 semantic checkpoint.

The strongest current results are:

- LoL3 runtime is clearly `WSX`-driven.
- `QWOD0` and `SWOD4` are the two true semantic anchors.
- `CKEEP` is a real hybrid feeder, but not a third semantic anchor.
- the outer layer is best described as a shared typed-region / regime-path system.
- the byte-level semantic checkpoint is now strong enough to promote:
  - `QWOD0`: `ad -> c5 -> a5`, with `eb` as mixed side branch
  - `SWOD4`: early `0b -> 02 -> 06`, then collapse into `0b -> 00` and `00 -> 00`

What this means:

- the LoL3 lane is no longer “just structurally constrained”
- it now has durable byte-level semantic footholds
- the remaining work is mostly meaning refinement above those footholds
